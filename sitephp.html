Site Web de nouvelles et de personnages avec PHP

Description du site

Site de présentation de nouvelles et de personnages de jeux-vidéos ou l'on affiche les nouvelles par catégories. Il est aussi possible de gérer les nouvelles en les modifiant, supprimant ou tout simplement en ajouter des nouvelles.

Année de conception

2021

Cours

Infrastructure WEB

Description du projet

Le site de nouvelles fût mon projet final de mon cours de PHP. Ce fût un très bon défi car j'ai dû mettre en pratique toutes les connaissances apprise durant le cours. Création de base donnée avec MySQL, manipulation de la base de donnée avec le CRUD. La manipulation apprise a été tant du coté du serveur ou sur un site avec PHP.
